# Hello World Java Program

A simple Java Hello World program.

## Getting Started

### Prerequisites
- Java Development Kit (JDK) 8 or higher

### How to Compile and Run

1. Compile the program:
   ```bash
   javac src/HelloWorld.java
   ```

2. Run the program:
   ```bash
   java -cp src HelloWorld
   ```

## Output
```
Hello, World!
```
